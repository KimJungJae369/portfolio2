나만의 가계부

데이터 분석으로 스마트한 소비 습관을 만드는 가계부 어플리케이션

프로젝트 소개

  • 사용자의 수입/지출을 직관적으로 관리하고 통계 데이터로 소비 패턴을 분석할 수 있는 웹 앱입니다.
  • 복잡한 가계부 관리를 단순하고 즐거운 경험으로 바꾸는 데 초점을 맞췄습니다.

주요 기능

  • 수입/지출 유형, 카테고리, 금액, 날짜를 기록하는 스마트 입력 시스템
  • 총 수입, 총 지출, 잔액을 한눈에 보여주는 통합 자산 대시보드
  • 평균값 계산과 카테고리별 분석 리포트 제공
  • 소비 배경을 기록할 수 있는 메모 기능 탑재
