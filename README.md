# Portfolio

React와 TypeScript로 제작한 포트폴리오 웹사이트입니다.

## 기능

- 반응형 디자인
- 부드러운 스크롤 네비게이션
- Crimson Text 폰트 사용
- Hero, About, Projects, Contact 섹션

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 프리뷰
npm run preview
```

## 기술 스택

- React 18
- TypeScript
- Vite
- CSS3

## 커스터마이징

- 색상: `src/index.css`의 CSS 변수 수정
- 내용: 각 컴포넌트 파일 수정
- 프로젝트: `src/components/Projects.tsx`의 projects 배열 수정
